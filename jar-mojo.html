hpi:jar

Full name:

org.jenkins-ci.tools:maven-hpi-plugin:1.117:jar

Description:

Build a jar separate from the hpi goal. If you do not use this goal then the HpiMojo will generate a jar file anyway. Using this goal it is possible to customize the jar file that is built, e.g. shading dependencies, code signing, etc

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: runtime.
  • Since version: 1.115.
  • Binds by default to the lifecycle phase: prepare-package.
